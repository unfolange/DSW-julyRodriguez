Conceptos

Propiedades de texto

Propiedad Descripcion Valores
Color Establece el color de un texto RGB, hex, keyword
line-height Establece la distancia entre líneas normal normal, number, length, %
letter-spacing Aumentar o disminuir el espacio entre caracteres. normal, length
text-align Alinea el texto en un elemento left, right, center, justify
text-decoration Agrega decoración al texto none, underline, overline, line-through
text-indent Agrega sangria a la primera linea de texto en un elemento length, %
text-transform Controla las letrast none, capitalize, uppercase, lowercase

Normas apa: Propiedades de texto. (2020). w3schools.com. https://www.w3schools.com/css/css_text.asp

Para más información visite la página w3schools.com CSS Text.

Propiedades de lista

Propiedad Descripcion Valores
list-style Establece todas las propiedades de una lista en una declaración list-style-type, list-style-position, list-style-image, inherit
list-style-image Especifica una imagen como marcador de elemento de lista. URL, none, inherit
list-style-position Especifica dónde colocar el marcador de elemento de lista inside, outside, inherit
list-style-position Especifica el tipo de marcador de elemento de lista none, disc, circle, square, decimal, decimal-leading-zero, armenian, georgian, lower-alpha, upper-alpha, lower-greek, lower-latin, upper-latin, lower-roman, upper-roman, inherit

Normas apa: Propiedades de Listas. (2020). w3schools.com. https://www.w3schools.com/css/css_list.asp

Para más información visite la página w3schools.com CSS List.

Propiedades de bordes

Propiedad Descripcion Valores
border Establece todas las propiedades del borde en una declaración border-width, border-style, border-color
border-bottom Establece todas las propiedades del borde inferior en una declaración border-bottom-width, border-bottom-style, border-bottom-color
border-bottom-color Establece el color del borde inferior border-color
border-bottom-style Establece el estilo del borde inferior border-style
border-bottom-width Establece el ancho del borde inferior border-width
border-color Establece el color de los cuatro bordes. color_name, hex_number, rgb_number, transparent, inherit
border-left Establece todas las propiedades del borde izquierdo en una declaración border-left-width, border-left-style, border-left-color
border-left-color Establece el color del borde izquierdo border-color
border-left-style Establece el estilo del borde izquierdo border-style
border-left-width Establece el ancho del borde izquierdo border-width
border-right Establece todas las propiedades del borde derecho en una declaración border-right-width, border-right-style, border-right-color
border-right-color Establece el color del borde derecho border-color
border-right-style Establece el estilo del borde derecho border-style
border-right-width Establece el ancho del borde derecho border-width
border-style Establece el estilo de los cuatro bordes. none, hidden, dotted, dashed, solid, double, groove, ridge, inset, outset, inherit
border-top Establece todas las propiedades del borde superior en una declaración border-top-width, border-top-style, border-top-color
border-top-color Establece el color del borde superior border-color
border-top-style Establece el ancho del borde superior border-style
border-top-width Establece el ancho del borde superior border-width
borders-width Establece el ancho de los cuatro bordes. thin, medium, thick, length, inherit

Para más información visite la página w3schools.com CSS Borders.

Propiedades de fuente

Propiedad Descripcion Valores
font Establece todas las propiedades de la fuente en una declaración font-style, font-variant, font-weight, font-size/line-height, font-family, caption, icon, menu, message-box, small-caption, status-bar, inherit
font-family Especifica la familia de fuentes para el texto. family-name, generic-family, inherit
font-sizer Especifica el tamaño de fuente del texto xx-small, x-small, small, medium, large, x-large, xx-large, smaller, larger, length, %, inherit
font-style Especifica el estilo de fuente del texto. normal, italic, oblique, inherit
font-variant Especifica si un texto debe mostrarse en letra pequeña normal, small-caps, inherit
font-weight Especifica el peso de una fuente normal, bold, bolder, lighter, 100, 200, 300, 400, 500, 600, 700, 800, 900, inherit

Normas apa: Propiedades de fuentes. (2020). w3schools.com. https://www.w3schools.com/css/css_font.asp

Para más información visite la página w3schools.com CSS Fuentes.
